# abbreviate

Plugin jQuery que abrevia números extensos com seus devidos prefixos. 

Colaborações serão bem-vindas!

#### HTML

```html
<span class="abbreviate" />
```

#### Tipos de abreviação:

0 "0"

12 "12"

123 "123"

1234 "1.2K”

12345 "12K”

123456 "0.1M”

1234567 "1.2M”

12345678 "12M”

123456789 "0.1B”

1234567899 "1.2B”

12345678999 "12B”
